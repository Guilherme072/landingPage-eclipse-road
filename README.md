
# Eclipse Road - Website Oficial

Este é o site oficial da **Eclipse Road**, uma empresa de marketing dedicada a ajudar clientes a alcançar o máximo potencial de suas marcas. O site foi desenvolvido para apresentar os serviços da empresa, destacar seus clientes e criar uma presença digital profissional.

## 📜 Funcionalidades

- **Seção de Apresentação**: Informações sobre a empresa e seus serviços.
- **Seção de Clientes**: Um slideshow dinâmico com fotos e descrições dos clientes.
- **Responsividade**: O design se adapta a diferentes tamanhos de tela para oferecer uma ótima experiência em dispositivos móveis e desktops.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilização e design responsivo.
- **JavaScript**: Funcionalidades interativas, como o slideshow de clientes.

## 🎯 Objetivos

- Profissionalizar a presença digital da Eclipse Road.
- Facilitar o acesso às informações sobre os serviços oferecidos.
- Criar uma experiência visual atraente e interativa para os visitantes.
